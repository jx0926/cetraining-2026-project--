## 项目简介
本项目是一个基于 Python 的学生成绩管理系统，支持添加、删除、查询、排序学生成绩，并可将数据保存到本地文件。

## 核心功能列表
- 添加学生成绩（姓名 + 科目 + 分数）
- 查看所有学生成绩
- 计算总分和平均分
- 按分数从高到低排序
- 保存数据到 CSV 文件
- 从 CSV 文件读取数据
## 如何运行项目
1. 确保已安装 Python 3.6+
2. 克隆本仓库：`git clone <仓库地址>`
3. 进入目录：`cd cetraining-2026-project-成绩管理系统`
4. 运行程序：`python main.py`

## 功能演示说明
- 输入 `1` 添加学生成绩
- 输入 `2` 查看所有成绩及总分/平均分
- 输入 `3` 查看成绩排名
- 输入 `4` 保存数据到 CSV 文件
- 输入 `5` 退出系统
